# cvimg

画像ファイルのフォーマットを変換する．

## Usage

```sh
    chmod +x ./cvimg.sh
```

```sh
    ./cvimg.sh [command] <input file or directory> <output directory>
```

## Help

```sh
    If <output directory> is not specified, the file is saved in the current directory.

    Available Commands:
        -h              print this.
        -f [FORMAT]     specify format. [FORMAT]：[png, jpg, pdf, svg, icns]
```
